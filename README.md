🛒 Supermarket Sales Data Analysis Project (Tableau)
📝 Overview
This project is a Supermarket Sales Data Analysis Dashboard built using Tableau.
It focuses on analyzing sales performance, customer behavior, and product trends to improve business decisions and increase profitability in retail operations.
🎯 Objectives
Analyze overall supermarket sales performance
Identify top-selling products and categories
Track revenue, profit, and quantity trends
Understand customer purchasing behavior
Improve sales strategy and inventory planning
🛠️ Tools Used
Tableau
Excel (Data Cleaning & Preparation)
Data Visualization
Basic Data Analysis
📊 Dashboard Features
Interactive filters (City, Branch, Product Line, Date)
KPIs (Total Sales, Profit, Quantity Sold, Average Rating)
Sales breakdown by product category
Customer segmentation analysis
Time-based sales trends
Clean and interactive dashboard design
🔍 Key Insights
Certain product categories generate higher revenue and profit
Sales performance varies across branches and cities
Customer preferences differ based on product type
Peak sales periods can be identified for better planning
📁 Files Included
Tableau Dashboard (.twbx / published link)
Dataset (Excel file)
README file
🚀 Conclusion
This project demonstrates how Tableau can be used in retail and supermarket analytics to understand sales performance, customer behavior, and product trends, enabling data-driven business decisions.
